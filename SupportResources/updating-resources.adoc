Updating Resources
------------------

Co-op Support's resources for libraries are constantly being being updated and expanded.

Tier 1 Resource Tasks
~~~~~~~~~~~~~~~~~~~~~

Tier 1 should:

* correct obvious typos found in the https://docs.libraries.coop/sitka/[Sitka], 
https://docs.libraries.coop/acquisitions/[Acquisitions], and https://docs.libraries.coop/policy/[Policy] manuals
* correct obvious formatting issues found in the https://docs.libraries.coop/sitka/[Sitka], 
https://docs.libraries.coop/acquisitions/[Acquisitions], and https://docs.libraries.coop/policy/[Policy] manuals
* update documentation sections as assigned
* xref:_reporting_issues_with_resources[report] any documentation that is outdated or contains incorrect information and is beyond the scope of what they have 
been assigned to correct or update themselves
* xref:_reporting_issues_with_resources[report] any links that are broken in any Co-op Support resource
* xref:_reporting_issues_with_resources[report] any other resources, such as https://bc.libraries.coop/sitka-faqs/[FAQs] 
and https://www.youtube.com/channel/UCiMYBCQG4QJVT-B3Ruk0Ncg/[videos], that are outdated or contain incorrect information.

Reporting Issues With Resources
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Co-op Support strives to keep our documentation up to date. 

Outdated or incorrect content and screenshots found in any of our manuals or other resources should be reported so 
that it can be fixed.

. Go to the *Support Docs* plan in Planner.
. Create a task in the appropriate bucket.
.. Task Name = name of the doc section, video, FAQ
.. Labels = add the appropriate labels
.. Notes = add a link to the existing documentation and an explanation of what the issue is
.. Assign to the appropriate staff member
. Click the close button (X).
. The assigned staff member will be alerted that the task has been assigned to them.
