Record Loading
--------------

Tickets with bibliographic or patron records to be loaded

.Process
. Ticket is received
. Move to Sitka Record Loading Queue
. If it's Patron records = assign to Support Staff
. If it's Bibliographic records = leave unassigned
. Set ticket priority = Low
. Add Library tag
. Add Sitka Record Loading tag 
.. Bibliographic records = based on record source
.. Patron records = Patrons
. Add Library main RT account to Requestor

Patron records will be loaded immediately

Bibliographic records will be loaded on Wed & Friday
